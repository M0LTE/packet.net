# json-spec

Generated JSON dumps of the SDL specification. Each `*.g.json` corresponds
to one `spec-sdl/**/*.sdl.yaml`. The shape is described by `schema.json`
(JSON Schema draft-2020-12); the codegen validates every emission against
the schema before writing.

Consumers can read these files in any language without writing a custom
emitter — JS, Python, R, jq scripts, etc. See `index.json` for the
manifest of available pages.

This directory is generated by `tools/Packet.Sdl.CodeGen`. Do not
hand-edit any of the `*.json` files — they are overwritten on every run.
To regenerate:

```sh
dotnet run --project tools/Packet.Sdl.CodeGen -- --json
```
